Front-end is done using HTML,CSS and Javascript. 
Backend is done using Node.js and express.js
Download the Assignment: First, download the assignment files to your computer.
Install npm: install npm.
Install Required npm Packages: The required packages are express,cors and winston. Open a terminal and run this command 
npm -i express,cors, winston.
